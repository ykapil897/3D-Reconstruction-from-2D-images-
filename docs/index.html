3D Reconstruction with Uncalibrated Stereo

A computer vision project for 3D reconstruction from uncalibrated stereo images.

View on GitHub See Results

Problem Statement

As a drone delivery company, we face challenges in efficiently navigating urban environments with complex structures and obstacles. Traditional 2D mapping techniques may not provide sufficient situational awareness, hindering safe and precise delivery operations.

Our 3D reconstruction solution accurately models the environment, enabling drones to plan optimal flight paths, detect obstacles, and ensure reliable deliveries in densely populated areas.

Our Approach

  1. Image Acquisition: Capture multi-view images around an object
  2. Stereo Rectification: Align image pairs for correspondence
  3. Disparity Map Calculation: Compute pixel-wise disparities
  4. 3D Reprojection: Convert 2D disparities to 3D points
  5. Point Cloud Alignment: Merge point clouds using ICP
  6. Surface Reconstruction: Create a 3D mesh

Technical Implementation

Key Features

  • Uncalibrated multi-view stereo reconstruction
  • Iterative Closest Point (ICP) alignment
  • Adaptive Semi-Global Block Matching
  • Point cloud filtering and optimization
  • Interactive web interface with Streamlit

Technologies Used

  • Python
  • OpenCV
  • Open3D
  • NumPy
  • Matplotlib
  • Streamlit
